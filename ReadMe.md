# A simple WebRTC video chat app with node and express

## Install denpendencies:
```bash
npm i express ejs socket.io uuid
npm i --save-dev nodemon

sudo npm i -g nodemon
sudo npm i -g peer

```

## How to run
```bash
[terminal 1]
peerjs --port 3001

[terminal 2]
nodemon server.js
```

[browser]
http://localhost:3000
